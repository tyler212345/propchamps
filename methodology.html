Weekly audit process

What we check, every Monday.

Each firm we cover goes through the same five-step audit at the start of every week. If something changed, we publish the diff within 72 hours.

01
Rule docs

Published rules pulled and diffed.

The firm's rules page is captured and compared against last week's snapshot. Anything that changed — payouts, drawdowns, news restrictions, consistency requirements — is flagged.

72hTo flag a rule change
02
Checkout

CHAMP code re-tested at checkout.

The CHAMP discount code is tested at checkout on every account size the firm offers. If a code fails for any tier, it's removed from the deals page within 24 hours.

100%Codes verified before publishing
03
Friction

Trader-reported issues logged.

A curated set of Discord and Reddit threads is monitored for active-trader reports — payout delays, support issues, surprise rule enforcement. Patterns get logged with sources.

14Communities monitored
04
Plan data

Pricing & drawdowns refreshed.

Account-size pricing, profit splits, drawdown amounts, and platform fees are pulled fresh. Any updates propagate to the firm card, comparison matrix, and full review automatically.

11Rule fields tracked per plan
05
Publish

Changes pushed to the changelog.

Rule diffs get dated entries on the firm's changelog and a one-line summary on its firm card. Material changes (new program, payout method change) get a full revision note.

Ranking criteria

How we rank.

Our weekly rank is a composite score across four weighted criteria. Affiliate revenue is not one of them. Firms cannot pay to move up.

Composite weighting

30 + 30 + 20 + 20 = 100. Each firm gets a score 0–100. The rank you see is sorted on that score.

Rule transparency
30%

How clearly the firm publishes its drawdown method, payout cadence, consistency rules, and news restrictions. Hidden rules drop the score.

Payout reliability
30%

Trader-reported payout speed against the firm's published cadence. Delays without communication drop the score.

Active discount value
20%

How much the active CHAMP discount actually saves a trader on the most-purchased account sizes. Verified weekly at checkout.

Program flexibility
20%

Number of paths to live capital (one-day pass, evaluation, instant funding) and how clearly each path is differentiated.

About PropChamps

Why this site exists.

PropChamps started because picking a futures prop firm means comparing dozens of rules across firms whose terms change without warning. The existing comparison sites we tried were either out of date by the time we used them, or read like affiliate funnels rather than research.

Our goal: build the reference traders open when they want to know what's actually true about a firm this week. We track rules, drawdowns, payouts, and promo codes — and we publish what changes.

Affiliate disclosure

How we make money.

PropChamps earns commission when readers click an affiliate link on this site and sign up with a firm. The CHAMP discount code is itself an affiliate code — using it gets you a discount, and the firm credits us a percentage of your purchase. Standard arrangement for comparison sites.

What affiliate revenue does not do:

  • Affect rankings. Firms pay zero to be ranked higher. Our composite score is calculated from the four criteria above — never from commission rates.
  • Hide negative information. If a firm tightens rules or delays payouts, it goes on the page next to the affiliate link.
  • Block coverage. We list firms whose affiliate programs we are not part of when they're relevant.
In one sentence

If a firm pays us more, it doesn't move up. If a firm pays us nothing, it doesn't move down. The CHAMP code is a real discount and we earn from it — that's the entire financial relationship.
