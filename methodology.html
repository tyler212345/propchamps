Methodology · Updated weekly

How we cover prop firms.

PropChamps is independent research on futures prop trading firms. Every firm we cover is audited weekly. Here's exactly what we check, how we rank, and how affiliate revenue works.

Weekly audit process

What we check, every Monday.

Each firm we cover goes through the same five-step audit at the start of every week. If something changed, we publish the diff within 72 hours.

/ 01

Rule documentation pulled

We pull the firm's published rules page and compare it against last week's snapshot. Anything that changed — payouts, drawdowns, news restrictions, consistency requirements — is flagged.

/ 02

Checkout price verification

The CHAMP discount code is tested at checkout on every account size offered. If a code fails for any tier, it comes off the deals page within 24 hours.

/ 03

Trader-reported friction

We monitor a curated set of Discord and Reddit threads where active traders report payout delays, support issues, or surprise rule enforcement. Patterns get logged.

/ 04

Plan data refreshed

Account-size pricing, profit splits, drawdown amounts, and platform fees are pulled fresh. Any updates propagate to the firm card, comparison matrix, and full review automatically.

/ 05

Changes published

Rule diffs get dated entries on the changelog and a one-line summary on the firm card. Material changes (new program, payout method change) get a full revision note.

Ranking criteria

How we rank.

Our weekly rank is a composite score across four criteria. Affiliate revenue is not one of them. Firms cannot pay to move up.

Rule transparency
30%

How clearly the firm publishes its drawdown method, payout cadence, consistency rules, and news restrictions. Hidden rules drop the score.

Payout reliability
30%

Trader-reported payout speed against the firm's published cadence. Delays without communication drop the score.

Active discount value
20%

How much the active CHAMP discount actually saves a trader on the most-purchased account sizes. Verified weekly at checkout.

Program flexibility
20%

Number of paths a trader has to live capital (one-day pass, evaluation, instant funding) and how clearly each is differentiated.

About PropChamps

Why this site exists.

PropChamps started because picking a futures prop firm involves comparing dozens of rules across firms whose terms change without warning. The existing comparison sites we tried were either out of date by the time we used them, or read like affiliate funnels rather than research.

Our goal: build the reference traders open when they want to know what's actually true about a firm this week. We track rules, drawdowns, payouts, and promo codes — and we publish what changes.

If you find an error or a firm we should be covering, email hello@propchamps.net.

Affiliate disclosure

How we make money.

PropChamps earns commission when readers click an affiliate link on this site and sign up with a firm. The CHAMP discount code is itself an affiliate code — using it gets you a discount, and the firm credits us a percentage of your purchase. Standard arrangement for comparison sites.

What affiliate revenue does not do:

  • Affect rankings. Firms pay zero to be ranked higher. Our weekly composite score is calculated from the criteria above, not commission rates.
  • Hide negative information. If a firm tightens rules or delays payouts, that goes on the page next to the affiliate link.
  • Block coverage. We list firms whose affiliate programs we are not part of when they're relevant.
In one sentence

If a firm pays us more, it doesn't move up. If a firm pays us nothing, it doesn't move down. The CHAMP code is a real discount and we earn from it — that's the whole financial relationship.
